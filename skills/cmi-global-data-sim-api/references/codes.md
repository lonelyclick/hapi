# 中国移动国际全球数据卡平台 API (V4.2) 状态码与返回码

说明：本文件摘录原文“4 HTTP 状态码 / 5 通用返回码”。

4 HTTP 状态码
| 分类 | 状态码 | 名称 | 说明 |
| --- | --- | --- | --- |
| GET | 200 | OK | 获取资源成功。 |
| GET | 302 | FOUND | 重定向响应。可以直接通过GET方法获取定位响应消息头中URI对应 |
| GET | 500 | SERVER ERROR | 服务内部错误。 |
| POST | 200 | OK | 更新资源成功。 |
| POST | 201 | CREATED | 新增资源成功。 |
| POST | 400 | BAD REQUEST | 参数错误，或参数格式错误。 |
| 通用 | 401 | UNAUTHORIZED | 鉴权失败。或所请求资源不属于对应账号不属于鉴权账号的 |
| 通用 | 404 | NOT FOUND | 资源找不到。 |
| 通用 | 405 | METHOD NOT ALLOWED | 资源对应该方法不支持。 |
| 通用 | 429 | TOO MANY REQUESTS | 同一时间请求数过多。 |
| 通用 | 500 | SERVER ERROR | 服务器内部错误。 |
